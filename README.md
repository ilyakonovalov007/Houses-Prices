# 🏠 House Price Prediction — ML Pet Project

### Зачем этот проект?
Рынок недвижимости — один из самых сложных для оценки: цена дома зависит от десятков факторов, и даже опытный агент ошибается на 15–20%. Я хотел проверить, может ли ML делать это точнее.

### Что я сделал
Взял датасет Ames Housing (Kaggle), выбрал 8 ключевых признаков на основе корреляционного анализа и сравнил 4 модели:

| Модель | MAPE |
| :--- | :--- |
| Linear Regression | ~15.3% |
| Ridge / Lasso | ~15.3% |
| Random Forest | ~12.0% |
| **CatBoost + log-таргет** | **~10.6% ✅** |

### Ключевые решения, которые улучшили результат
* **Корреляционный отбор признаков** — не брал всё подряд, а выбрал только то, что реально влияет на цену (`OverallQual`, `GrLivArea`, `GarageCars` и др.).
* **Логарифмирование таргета** — цены на дома имеют правый хвост, log-трансформация сделала распределение нормальным и снизила MAPE с 10.76% до 10.59%.
* **CatBoost с категориальным признаком** — добавил `Neighborhood` напрямую без label encoding, что дало прирост точности против Random Forest.

### Итог
Финальная модель ошибается в среднем на **~$17 000** при средней цене дома **$180 000** — это точнее, чем среднестатистический агент по недвижимости.
